# TykoTech Fork User Guide

Welcome to TykoTech Fork! This guide explains how to install and use the application.

## Installation

1. Download the latest release from our [GitHub Releases](https://github.com/TykoTech/TykoTech-Fork/releases) page.
2. Extract the archive for your operating system (Windows, macOS or Linux).
3. Run the provided executable to start the application.

### Build from Source

If you prefer to build the application yourself:

```bash
corepack enable
corepack prepare yarn@4.6.0 --activate
yarn install
# Build for your current platform
yarn build:current_os
```

## Basic Usage

- Create new conversations with any of the built‑in AI assistants.
- Manage your documents and knowledge base using the sidebar tools.
- Configure settings such as themes or API keys from the **Settings** menu.

For advanced features and troubleshooting, check the other documents in the `docs` folder.
